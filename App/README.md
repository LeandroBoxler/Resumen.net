# Resumenes.net App

Aplicación móvil de resúmenes de estudio desarrollada con Flutter.

## Requisitos

- Flutter SDK 3.0.0 o superior
- Dart 3.0.0 o superior

## Instalación

```bash
flutter pub get
```

## Ejecución

```bash
flutter run
```

## Configuración de la API

Edita la URL base de la API en `lib/services/api_service.dart`:

```dart
static const String baseUrl = 'tu-api-url';
```
